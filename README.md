# To-Do List App 🎉

**A simple and elegant To-Do List application built with HTML, CSS, and JavaScript.** 📜

This project provides a user-friendly interface for managing your daily tasks.  You can add, delete, mark as complete, and even drag-and-drop to reorder your tasks! 🤝

## Installation 💾

1.  Save the code as an HTML file (e.g., `todo.html`).
2.  Optionally, create a `favicon.ico` file and place it in the same directory.


## Features ✨

*   **Add Tasks:** Easily add new tasks to your list.
*   **Mark as Complete:** Check off tasks as you finish them.
*   **Delete Tasks:** Remove completed or unwanted tasks.
*   **Drag-and-Drop Reordering:**  Reorder tasks by dragging and dropping them.
*   **Task Counter:** View the number of pending tasks.
*   **Dark Mode:** Toggle between light and dark themes. 🌚🌞
*   **Local Storage:** Tasks are saved locally, so they persist even after closing the browser.


## Usage Instructions ▶️

1.  Open the `todo.html` file in your web browser.
2.  Enter a task in the input field.
3.  Click "Add Task" to add it to the list.
4.  Click on a task to mark it as complete or incomplete.
5.  Click the "Delete" button to remove a task.
6.  Drag and drop tasks to reorder them.
7.  Click the "light🌞/Dark🌚" button to switch between light and dark modes.


## Dependencies 🔧

This project has no external dependencies. It uses only built-in HTML, CSS, and JavaScript features.


## Commands to Run the Project ⚙️

No special commands are needed.  Simply open the `todo.html` file in your web browser.
